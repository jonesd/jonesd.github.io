Genetic Programming Trynth

This was an attempt to create computer players for my Trynth space strategy game using Genetic Programming.

Multiplayer turn based space strategy game. Created initially as my project work for the Expert Systems course in university, it evolved into quite a comprehensive gamen give the limited mechanics of the game. Developed in BBC BASIC running under RiscOS. Over time I have used it as a test bed for playing around with some simple AI techniques for generating computer players. I also played around with Microsofts DirectX stuff by working on a realtime version of Trynth, with included an extended game mechanics.
