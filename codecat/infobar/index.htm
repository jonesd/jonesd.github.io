Info Bar

This is a simple tool for displaying critical readouts of the systems state in easy to access, but undominating, location of the screen. In the end I choose on the far left of the task bar, just at the top, so with a suitably small textfont it does not overlap with the drive icons. Implemented in assembly as a module for minimal memory consumption on RiscOS3.1.
