Searcher

0.01 (02 Nov 1994) David G Jones

This is a small unfinished app written in an afternoon to search through a number of files searching for a word.

The word should be supplied in lower case. This will then match with any combination of upper case/lower case in the supplied texts. Text searching is fast, but going through the directory trees is slow. At the moment the text is loaded fully into memory rather than in small chunks, so the resulting large wimp slot.
