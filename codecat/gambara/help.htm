Gambara

0.03 (14-Oct-1994) © David G Jones

This assembly application will simply present a window displaying information on the pointer. Specifically:

  • X & Y pointer positions
  • State of mouse buttons
  • Window under the pointer
  • Task which owns this window
  • Icon under the pointer

In the final three cases a name is associated with the numerical data if possible. The Task & Window numbers are in Hex, the rest decimal.

And thats about it really. Nothing earth shattering but what more could you ask for...

The customary set of ideas for future outlandish additions:

  • More in depth display of information, ie Icon flags. Though there are a number of good apps for that already.
  • Allow the delay between updating the display to be set.
  • Totally over the top detection of mouse buttons, so always informed of changes.
