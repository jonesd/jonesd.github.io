Multiplayer turn based space strategy game developed in BBC BASIC for the Acorn Archimedes in 1994.

  • Manual
  • Genetic Programming
  • Future

Created initially as my project work for the Expert Systems course in university, it evolved into quite a comprehensive game give the limited mechanics. Over time I have used it as a test bed for experimenting with some simple AI techniques for generating computer players. There was also a prototype in Microsofts DirectX of a real-time version of Trynth, which included extended game mechanics.
