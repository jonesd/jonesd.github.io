Method Stack Profiler

The New Profiler is a type of sampling profiler which records the details of the message stack.

Mechanics

The Method Stack Profiler is run after each sampling period and generates an entry for each frame of the profiled processes stack. An entry records the method, receiving class, ip and whether the frame is executing a Block.

Browsers

The raw collected details are visible in the Profile Browser. A statistical view is provided by the Message Statistics view. A hierarchical call tree generated from a series of method stacks is provided by the Message Tally Browser.
