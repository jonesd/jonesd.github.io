Strand Context

The Strand Context dialog displays information on the context in which a browsers data is from.

The dialog can be opened from the Context item of a browsers first menu.

Structure

The top section of the dialog displays data on the profile. This is the code whose execution was profiled, and the time at which profiling commenced.

The lower section displays data on the Strand whose details the browser is showing. This indicates the profiler which recorded the information, and the range of iterations which the browser is showing.
