Event Statistics

The Event Statistics view displays some simple statistics generated from the Event Profiler within the synchronised range at the time of opening the view.

The Message Statistics can be opened from the Strand menu of the Director. Any number can be opened and they are not reactive.

Structure

The statistics view presents a number of different views onto the same set of data collected by the profiler. Each pane groups the individual event triggerings in different combinations.

The Received column in each pane relates to the cumulative total of items which were activated in response to a trigger. The Sent column in each pane relates to the cumulative total of items that were triggered.

The status bar displays the totals for each item, from the Received and Sent columns.

Events Pane

The Events pane groups all the triggerings by the event Message selector.

Classes Pane

The Events pane groups all the triggerings by the Class of the receiving object and the events Message selector.
