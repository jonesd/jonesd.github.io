Window Dispatch Profiler

The Window DispatchProfiler is a type of activating profiler which records the dispatching of Windows messages to Dolphin views.

Mechanics

The New Profiler patches the dispatchMessage:wParam:lParam: method of View, which is the entry point into the View hierarchy of messages which have been dispatched by Windows. The Profiler is able to record the class of the receiving view and the identity of the message type. The identity is displayed as a matching symbol from Views MessageMap class variable in the browsers, or its number if there is no matching value.

Browsers

The raw collected details are visible in the Profile Browser. A statistical view is provided by the Window Dispatch Statistics view.
