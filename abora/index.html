Abora Hypertext System

Abora is an ongoing project to implement a hypertext system based on a Xanalogical Structure. It will support fine grained links, transclusion of content, user filtering of links and support client-server and server-server distribution. This project is released under the MIT open-source license and source code for Dolphin Smalltalk is included. An executable demo for the MS Windows platform is available.

The project home page can be found at: http://www.abora.org
